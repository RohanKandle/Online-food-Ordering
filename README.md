🍔 Foodie Hub - Online Food Ordering Web Application


📌 Overview

Foodie Hub is a simple and interactive online food ordering web application built using HTML, CSS, and JavaScript.

It allows users to:

Browse food items
Search food dynamically
Add items to cart
Update quantity
Place orders
View order history

All data is stored using browser localStorage, so data remains saved even after page refresh.

🎯 Project Objectives
Build a frontend food ordering system
Practice JavaScript DOM manipulation
Implement cart and order logic
Create a simple authentication system
Store and manage data using localStorage

🚀 Features
🍕 Food Menu System
Displays multiple food items dynamically
Each item includes image, price, and flavor options
Fully generated using JavaScript

🔍 Search Functionality
Search food items by name
Case-insensitive filtering
Instant results update

🛒 Cart System
Add items to cart
Same item with same flavor increases quantity
Remove item automatically when quantity reaches 0
Live cart counter and total price update

➕➖ Quantity Control
Increase quantity using "+" button
Decrease quantity using "-" button
Automatic total price recalculation

💳 Payment Simulation
Simple checkout form (Name, Address, Phone)
Order confirmation system
Success message after order placement

📦 Order History
Stores all previous orders in localStorage
Shows:
Customer details
Ordered items
Total price
Date & time

🔐 Authentication System (Frontend Only)
User Registration
User Login
Logout functionality
Data stored in localStorage

🚪 Logout Feature
Removes active user session
Redirects back to login screen

🛠️ Tech Stack
Technology	Purpose
HTML5	Structure
CSS3	Styling
JavaScript (Vanilla JS)	Logic & functionality
LocalStorage	Data storage
